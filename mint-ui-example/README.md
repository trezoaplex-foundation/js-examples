# Mint from candy machine through the browser

In this tutorial, we will see how to mint from candy machine with wallet in the browser using the [Trezoa Wallet Adapter Library](https://github.com/trezoa-labs/wallet-adapter) and the [Trezoaplex JS SDK](https://github.com/metaplex-foundation/js).

The Guide is available in the [official Trezoaplex docs](https://docs.trezoaplex.com/programs/candy-machine/how-to-guides/my-first-candy-machine-part2).
